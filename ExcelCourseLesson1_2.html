Mixed reference

We have learned the Absolute Reference. But let's imagine that you need to convert your values to three different currencies. Using the Absolute Reference you will need to enter the formula three times like in the example below:

sum 380 420 6
37500 =E3/$F$2 =E3/$G$2 =E3/$H$2
... ... ... ...

Then we need to copy each formula down so that all the values are converted to a respective currency.

Is there a way to accomplish this task with one formula? Yes there is (when working with a computer programm, we always need to eliminate something repeatetive). And the way to do that is to use Mixed Reference. We can say that there are two subtypes of Mixed Reference.

They are:

  • Constant Column
  • Constant Row

Let's see their difference on an example.
