Lesson 1. Reference types

There are three types of references in Excel. And then there is R1C1 reference style (I will be calling it just 'R1C1' from now on) vs 'usual one'. We will take a look on R1C1 a little bit later. So here are reference types:

  1. Relative
  2. Absolute
  3. Mixed
    • Constant Column
    • Constant Row

Relative reference

R1C1 reference style

In order to understand Relative reference we first need to look into R1C1 reference style . But to grasp what is R1C1, we need to go through some basics. So every cell in Excel Sheet has its own unique address, which consists of a combination of a letter and a digit. Letter represents a column, and digit represents a row. For example Cell A1 is located on intersection of the column A and the first row. Now if we switch to R1C1 (click here to see how it is done) we will see that now columns are in digits as well as rows.

Let's see it on an example. Below you can see the formula '=C3*D3', so we are multiplying price by quantity, and we are getting the sum.

Now if we change to R1C1 we will see this:

So what does the formula '=RC[-2]*RC[-1]' mean? Here 'R' stands for row, and as there is no digit after it, it means 'on the same row'. 'C' represents column, and it has '[-2]', so it means two columns to the left from the current cell. It is interesting that if we copy the formula down, it will appear the same in all the cells down (You can check it yourself in your Excel file). So we will see something like this:

quantity price sum
15 2500 =RC[-2]*RC[-1]
10 150 =RC[-2]*RC[-1]
10 160 =RC[-2]*RC[-1]
... ... ...

The reason that formula is the same in all the cells is that in all these cells we refer to a cell on the same row and two columns on the left multiplied by a cell on the same row and one column to the left.

So if we swith back to our 'normal' view (where columns are in letters) we will see this kind of formula:

quantity price sum
15 2500 =C3*D3
10 150 =C4*D4
10 160 =C5*D5
... ... ...

So how the formula '=RC[-2]*RC[-1]' can be equal to three different formulas:

  • =C3*D3
  • =C4*D4
  • =C5*D5
  • ...

One way to explain it is to say that Excel itself sees this formula in a R1C1 way, but shows us (i.e. humans) in a '=C3*D3' way. And that is the reason why this type of referencing is called 'Relative'. Because from the point of view of Excel, other cells which we refer to lie in a certain number of rows and columns.
