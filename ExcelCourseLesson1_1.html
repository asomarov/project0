Absolute reference

Now let's say we have a different task. We need to convert sums we have just calculated to some other currencies. The first currency rate is equal to 380.

Here I divide the cell E3 by the cell F2. But wait a minute, as I am using a relative reference here, I am not dividing it by the cell F2, I am dividing it by a cell, which is one row above from the curent cell. So if I copy this formula down I will get these formulas in the cells below the current:

sum 380
37500 =E3/F2
1500 =E4/F3
1600 =E5/F4
... ...

So now I am dividing a sum value by the wrong values. I need to make the reference to F2 cell constant. To do that you need to press .

And now you see dollar signs in the reference to F2 cell (i.e. $F$2):

And if you copy this formula down you will see that different sum values are divided only by the F2 cell:

sum 380
37500 =E3/$F$2
1500 =E4/$F$2
1600 =E5/$F$2
... ...

So as you see these dollar signs in the reference only show that your reference has changed to Absolute reference
